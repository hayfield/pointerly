Class Detail
Pointerly.Home(homeSetup)
A Home Position that resides in the middle of the screen
Parameters:
{object} homeSetup
The setup for the home position.
If undefined, setup is not completed.
Should have the following properties:
  • width
  • height
  • color
Field Detail
<inner> color
The color of the home area
Default Value:
If homeSetup.color is defined, uses that, otherwise orange
<inner> {number} height
The height of the home area
Default Value:
If homeSetup.height is a number, uses that, otherwise 40
<inner> {number} width
The width of the home area
Default Value:
If homeSetup.width is a number, uses that, otherwise 40
Method Detail
<inner> onHomeEnter()
A function which fires when the mouse enters the home area. It will call an onEnter() function if provided as a property of homeSetup.
<inner> onHomeExit()
A function which fires when the mouse exits the home area. It will call an onExit() function if provided as a property of homeSetup.
